# London Adventure 2025

A mobile-first travel guide presentation for a London trip (24 December 2025 - 1 January 2026).

## View

Open `index.html` in any browser, or visit the GitHub Pages site.

## Features

- 18 swipeable slides with day-by-day itinerary
- Booking priorities and practical notes
- Emergency contacts and toddler-friendly tips
- Mobile-optimized layout

## Tech

- Swiper.js for touch navigation
- Google Fonts (Playfair Display, Inter)
- Vanilla HTML/CSS (no build step)
